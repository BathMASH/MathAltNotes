4 Ordinary Differential Equations (ODEs)

Differential equations are widely used in science, engineering and economics. After 400 years there are still lots of open problems to solve.

4.1 Overview

A differential equation is an equation which links y , dy dx , d2y dx2 , etc.

Examples.

dy dx = xEasy
dy dx = x + y2 Hard
dy dx2 + dy dx = sin(y) Very hard
d2y dx2 + dy dx = sin(x) Easy

“Ordinary” means there is only one independent variable.

Some terminology

  • The order of a differential equation is the largest number of derivatives.
    dy dx = xFirst order
    d2y dx2 = x Second order
    d4y dx4 + pd2y dx2 + y = 0 Fourth order
  • A differential equation is linear if we only have d2y dx2 , dy dx or y and no functions such as y2 .
    • Linear equations are easy to solve and very common.
    • Nonlinear equations are often very hard to solve. The solutions usually use numerical methods.
  • The initial condition(s) of a differential equation is information about the solution at a point x = a . We need this to find a particular solution to the equation; that is, one that does not depend on any arbitrary constants.
    First order: Need to know y(a)
    Second order:Need to know y(a), dy(a) dx

4.2 Separable first order differential equations

Definition 4.1.

A separable first order ordinary differential equation has the form

dy dx = p(x)q(y).

We can solve these directly by integration:

Rearrange and integrate

1 q(y) dy dx = p(x) ⇒∫ 1 q(y) dy dxdx = ∫ p(x)dx ⇒∫ dy q(y) = ∫ p(x)dx

Example.

dy dx = xex2 y2

Separate and integrate

∫ y2dy = ∫ xex2 dx ⇒ y3 3 = 1 2ex2 + c ⇒ y3 = 3 2ex2 + c

Example.

In the introduction to the course we saw a basic model for world population. A more sophisticated model for population p(t) might satisfy the differential equation

dp dt = p(M − p).

Separating variables and integrating gives

∫ dp p(M − p) = ∫ dt LHS use partial fractions 1 p(M − p) = 1 M p + 1 M M − p ∴ 1 M ln p M − p = t + C.

To find C we need some initial data. Let p = p0 at t = 0 .

⇒ C = 1 M ln p0 M − p0 ∴ 1 M ln p M − p = t + 1 M ln p0 M − p0 .

Multiply by M and exponentiate

p M − p = p0 M − p0eMt.

Rearrange

p = p0 M − p0eMt M − p0 M − p0eMt p p 1 + p0 M − p0eMt = p0M M − p0 eMt

∴p = Mp0eMt M − p0 + p0eMt = Mp0eMt M − p0 ⋅ M − p0 M − p0 + p0eMt

Sketch:

At t = 0 ,

p = Mp0 M − p0 + p0 = p0.

As t →∞,

Mp0 p0 = M.

4.3 Homogeneous equations

A function f(x,y) is homogeneous of degree n if

f(tx,ty) = tnf(x,y).

Examples.

f(x,y) = x2 + xy ⇒f(tx,ty) = t2x2 + t2xy = t2f(x,y)

(homogeneous, degree 2 )

f(x,y) = x2 + y2 ⇒f(tx,ty) = tx2 + y2 = tf(x,y)

(homogeneous, degree 1 )

f(x,y) = sin y x

(homogeneous, degree 0 )

Degree 0 .

The general form for a homogeneous differential equation of degree 0 is

dy dx = f y x

Solution.  Let u = y x . Then

ux = y ⇒ u + xdu dx = dy dx.

Hence

u + xdu dx = f(u) ⇒ xdu dx = f(u) − u ⇒ ∫ du f(u) − u = ∫ dx x = ln x + c.

Example.

dy dx = y x 1 + ln y x

Let u = y x . Then

u + xdu dx = u(1 + ln(u)) = u + u ln(u) ⇒∫ du u ln(u) = ∫ dx x = ln x + c.

Let v = ln(u) . Then

dv du = 1 u(u > 0)

So

∫ du u ln(u) = ∫ dv v = ln v

Hence

ln v = ln(x) + c

If v > 0 , v = Ax , A = ec .

⇒ ln(u) = Ax ⇒ ln y x = Ax ⇒ u = xeAx.

4.4 Linear first order ordinary differential equations

These take the form

dy dx + p(x)y = q(x).

The LHS is a linear combination of y and its derivative.

We solve these by using an integrating factor.

Lemma 4.2.

If r(x) = e∫ p(x)dx then

dr dx = p(x)r(x).

Proof.  By chain rule

dr dx = e∫ p(x)dx d dx∫ p(x)dx = p(x)e∫ p(x)dx by Fundamental theorem of calculus = p(x)r(x)

□

The function r(x) = e∫ p(x)dx is called the integrating factor.

Proposition 4.3.

The equation

dy dx + p(x)y = q(x) (4.1)

has solution

y(x) = 1 r(x) ∫ r(x)q(x)dx.

Proof.  Consider

d dx(r(x)y(x)) = r(x)dy dx + ydr dx = r(x)dy dx + ypr = r(x) dy dx + yp

Hence if y satisfies (4.1), then

d dx(ry) = r dy dx + py = rq ⇒ y = 1 r∫ rqdx.

□

Examples.

1.
dy dx + y = x

So

dy dx + p(x)y = q(x),

where

p(x) = 1,q(x) = x.

Integrating factor:

r(x) = e∫ p(x)dx = ex.

Hence

y(x) = e−x ∫ exxdx = e−x[xex − ex + c] = x − 1 + ce−x.

(note: non-trivial constant)

2.
xdy dx + 2u = x3

Rearranging,

dy dx + 2y x = x2,

so

dy dx + p(x)y = qx,

where p(x) = 2 x , q(x) = x2 .

Integrating factor:

r(x) = e∫ 2 xdx = e2 ln(x) = x2.

Hence

y(x) = 1 x2 ∫ x2x2dx = 1 x2 x5 5 + c = x3 5 + c x2.

4.5 Bernoulli equations

A differential equation of the form

dy dx + p(x)y = q(x)yn,n ∈ ℤ

is called a Bernoulli equation (named after Jacob Bernoulli, who proposed it, and his brother Johann Bernoulli, who found a solution).

If n = 0 or 1 they are linear. Solve by integrating factor ( n = 0 ) or separation ( n = 1 ). Otherwise they are non-linear.

Method for n≠0 or 1 .

Idea: use a substitution to make the equation linear.

Divide by yn :

y−ndy dx + p(x)y1−n = q(x)

Let z = y1−n . Then by the chain rule

dz dx = (1 − n)y−ndy dx.

Substitute

1 1 − n dz dx + p(x)z = q(x)

This equation is now linear in the z -terms: use integrating factor to solve.

Examples.

 

1.
dy dx + 4 xy = x3y2

Bernoulli equation with

p(x) = 4 x,q(x) = x3,n = 2.

Divide by y2

y−2 dy dx + 4 xy−1 = x3

Let z = y−1 . So dz dx = −y−2 dy dx.

Substitute

−dz dx + 4 xz = x3 ⇒ dz dx − 4 xz = −x3

Integrating factor

r(x) = e∫ −4 xdx = e∫ −4 lnx = x−4.

Then

z(x) = 1 r(x) ∫ r(x)q(x)dx = x4 ∫ x−4(−x3)dx = x4 ∫ −x−1dx = x4(− ln x + c) = x4(c − ln x).

Hence

y = 1 x4(c − ln x).

2.
dy dx = 5y + e−2xy−2

Rearrange

dy dx − 5y = e−2xy−2

Bernoulli equation with

p(x) = −5,q(x) = e−2x,n = −2.

So z = y1−n = y3 satisfies

1 1 − n dz dx + p(x)z = q(x) ⇔ 1 3 dz dx − 5z = e−2x.

Rearrange

dz dx − 15z = 3e−2x

Integrating factor

r(x) = e∫ −15dx = e−15x

So

z(x) = e15x ∫ e−15x(3e−2x)dx = e15x − 3 17e−17x + c

Then

y = z1 3 = e5xc − 3 17e−17x3

3.
6dy dx − 2y = xy4

Rearrange

dy dx − y 3 = x 6y4

Bernoulli equation with

p(x) = −1 3,q(x) = x 6,n = 4.

So z = y1−n = y−3 satisfies

1 1 − n dz dx + p(x)z = q(x) ⇔ −1 3 dz dx − z 3 = x 6.

Rearrange

dz dx + z = −x 2

Integrating factor

r(x) = e∫ 1dx = ex

Hence

z(x) = e−x ∫ ex −x 2 dx = e−x −1 2(xex − ex) + c = ce−x − 1 2(x − 1)

Then y = z−1 3 .

4.
Logistic growth: (e.g. population) dy dx = y(M − y) (see Section 4.2)

Rearrange

dy dx − My = −y2

to Bernoulli equation with

p(x) = −M,q(x) = −1,n = 2.

So z = y1−n = y−1 satisfies

dz dx + Mz = 1

Integrating factor r(x) = e∫ Mdx = eMx gives

z(x) = e−Mx ∫ eMxdx = e−Mx eMx M + c = 1 M + ce−Mx ⇒y = z−1 = 1 M + ce−Mx −1 = M 1 + c̃e−Mx.

4.6 Exact equations

Consider a differential equation of the form

M(x,y) + N(x,y)dy dx = 0.

This includes functions of two variables x and y .

Suppose there exists some function f(x,y) such that

∂f ∂x = M

(“partial derivative”: rate of change with respect to x if y held constant)

∂f ∂y = N

(rate of change with respect to y if x held constant)

Then

∂f ∂x + ∂f ∂y dy dx = M + N dy dx = 0

Now, the chain rule for two variables says:

df dx = ∂f ∂x + ∂f ∂y dy dx

So

∂f ∂x + ∂f ∂y dy dx = 0 ⇒ df dx = 0 ⇒ f(x,y) = c.

So f(x,y) = c is an implicit solution to the differential equation, i.e. if x,y satisfy f(x,y) = c , they will also satisfy the ODE.

Test for exactness

The equation

M(x,y) + N(x,y)dy dx = 0

is exact if ∂M ∂y = ∂N ∂x .

Finding the solution f(x,y)

We can find f by integrating M or N , but we have to be careful. We have

∂f ∂x = M.

Integrate both sides with respect to x :

f(x,y) = ∫ Mdx + g(y),

where g(y) is constant with respect to x , but may depend on y .

What is g ? Differentiate both sides with respect to y :

∂f ∂y = ∂ ∂y∫ Mdx + g′(y).

Rearranging,

g′(y) = ∂f ∂y − ∂ ∂y∫ Mdx = N − ∂ ∂y∫ Mdx.

Hence

g(y) = ∫ g′(y)dy = ∫ N − ∂ ∂y∫ Mdxdy

So an exact ODE has implicit solution f(x,y) = c where

f(x,y) = ∫ Mdx + ∫ N − ∂ ∂y∫ Mdxdy

Examples.

1.
y + xdy dx = 0

Inspiration:

∂ ∂y(xy) = x ∂ ∂x(xy) = y

So f(x,y) = xy satisfied

∂f ∂x = M,∂f ∂y = N

where M = y , N = x .

Hence solution is xy = c .

2.
ey + (xey + 2y)dy dx = 0 .

So

M = ey,N = xey + 2y ⇒ ∂M ∂y = ey,∂N ∂x = ey.

Hence the ODE is exact. So solution is f(x,y) = c where

f(x,y) = ∫ Mdx + ∫ N − ∂ ∂y∫ Mdxdy = ∫ eydx + ∫ xey + 2y − ∂ ∂y∫ eydxdy = xey + ∫ xey + 2y − xeydy = xey + ∫ 2ydy = xey + y2

Hence implicit solution of ODE is

xey + y2 = c.

3.
2xy − 9x2 + (2y + x2 + 1)dy dx = 0

So

M = 2xy − 9x2,N = 2y + x2 + 1 ∂M ∂y = 2x,∂N ∂x = 2x

Hence exact.

Next, find f(x,y) :

∫ Mdx = ∫ 2xy − 9x2dx = x2y − 3x3

So

f(x,y) = x2y − 3x3 + ∫ 2y + x2 + 1 − ∂ ∂y(x2y − 3x3)dy = x2y − 3x3 + ∫ 2y + x2 + 1 − x2dy = x2y − 3x3 + ∫ 2y + 1dy = x2y − 3x3 + y2 + y = y2 + (x2 + 1)y − 3x3

Hence the ODE has implicit solution

y2 + (x2 + 1)y − 3x3 = c.

4.
2xy2 + 4 − 2(3 − x2y)dy dx = 0

So

M = 2xy2 + 4,N = −2(3 − x2y) ∂M ∂y = 4xy,∂N ∂x = 4xy

Hence exact.

Next, find f(x,y) .

∫ Mdx = ∫ 2xy2 + 4dx = x2y2 + 4x

Hence

f(x,y) = x2y2 + 4x + ∫ −2(3x − x2y) − ∂ ∂y(x2y2 + 4x)dy = x2y2 + 4x + ∫ −2(3 − x2y) − 2x2ydy = x2y2 + 4x −∫ 6dy = x2y2 + 4x − 6y.

Hence the ODE has implicit solution

x2y2 + 4x − 6y = c.

4.7 Linear second order ordinary differential equations

Overview

These take the form

ad2u dx2 + bdu dx + cu = 0(Free/Unforced) ad2u dx2 + bdu dx + cu = f(x)(Forced)

Example (Falling ball).

md2y dt2 + kdy dt = −gm.

Each of these equations has an uncountable number of solutions. To find a unique solution we must specify two properties of the solution.

A.
Initial value problem: u(0) = α , u′(0) = β ; u and u′ at some point,

e.g. throwing a ball.
B.
(Dirichlet) Boundary value problem: u(0) = α , u(1) = β ;

e.g. making the ball land on target.
C.
(Neumann) Boundary value problem: u′(0) = α , u′(1) = β ;

i.e.  u′ at two points, arises in studying biological problems.

Free linear second order ODEs

Theorem 4.4 (Superposition).

 Let p(x) and q(x) be solutions to the second order ODE

ad2u dx2 + bdu dx + cu = 0.

Then

r(x) = Ap(x) + Bq(x)

is also a solution for any constants A,B .

Proof.  Due to the linearity of the derivative

r = Ap + Bq ⇒ r′ = Ap′ + Bq′ ⇒ r″ = Ap″ + Bq″.

Hence

ar″ + br′ + cr = A[ap″ + bp′ + cp] + B[aq″ + bq′ + q] = 0 + 0 (since p.q solutions) = 0,

so r is also a solution to the equation. □

Definition 4.5.

We call r = r(x) a linear combination of p and q .

Definition 4.6.

 p and q are linearly independent if there are no non-zero constants A and B such that

Ap(x) + Bq(x) = 0∀x

Remark.

 Ap + Bq = 0 may still be satisfied at some values of x .

Theorem 4.7.

A free linear second order ODE has exactly two linearly independent solutions. Any solution of the ODE are linear combinations of these two solutions.

Definition 4.8.

Let the free linear second order ODE have two linearly independent solutions p(x) , q(x) . Then the general solution is

Ap(x) + Bq(x),  A,B ∈ ℝ  constants.

Example.

Simple harmonic motion (e.g. oscillation of a spring, simple pendulum, molecular vibration)

d2u dx2 + u = 0

Two linearly independent solutions are sin(x) and cos(x) .

So, general solution is

u = A cos(x) + B sin(x).

Constructing the General solution

Consider

ad2u dx2 + bdu dx + cu = 0 (4.2)

Suppose u = eλx , λ constant. Then

ad2u dx2 + bdu dx + cu = (aλ2 + bλ + c)eλx

Hence (4.2) is satisfied if λ is such that

aλ2 + bλ + c = 0

This quadratic in λ is the auxiliary equation (AE).

Solutions are

λ1,2 = −b ±b2 − 4ac 2a

Depending on the type of solutions of the AE there are three cases to distinguish:

Case 1.
( b2 − 4ac > 0 ) λ1,λ2 real and distinct

General solution

u = Aeλ1x + Beλ2x

Solution behaviour: No oscillations;

long term growth ( λ1 > 0 or λ2 > 0 ) or decline ( λ1 < 0 and λ2 < 0 ).

Remark.

 

eλ1x and eλ2x are linearly independent because

eλ1x eλ2x = e(λ1−λ2)x

is not constant.

Case 2.
( b2 − 4ac < 0 ) λ1,λ2 complex conjugates, λ1,2 = α ± iβ

General solution

u = eαx(A cos(βx) + B sin(βx)).

Solution behaviour: Oscillatory with long term growth (α > 0) or decline (α < 0) .

Proof. 

Real and imaginary parts of the roots of the auxiliary equation

λ1,2 = α ± iβ

are

α = − b 2a,β = 4ac − b2 2a .

Thus

u1,2 = e(α±iβ)x

are solutions of (4.2).

But u1 and u2 involve complex numbers for all x . The ODE is real-valued. We want real-valued solutions.

By Euler’s formula

u1,2 = eαx[cos(βx) ± i sin(βx)]

Recall superposition (Theorem 4.4): Linear combinations of u1 and u2 are also solutions (even with complex coefficients).

So

1 2u1 + 1 2u2 = eαx cos(βx)

is a solution, as well as

1 2iu1 − 1 2iu2 = eαx sin(βx)

is a solution.

Clearly eαx cos(βx) , eαx sin(βx) are real-valued and linearly independent.

So the general solution is

Aeαx cos(βx) + Beαx sin(βx).

□

Case 3.
( b2 = 4ac ) λ1 = λ2 , so write λ for the unique root of the AE.

Then λ = − b 2a is real-valued and the general solution is

u = Aeλx + Bxeλx.

Solution behaviour: no oscillations, long term growth ( λ > 0 ) or decline ( λ < 0 ).

Proof.  If

u = Aeλx + Bxeλx

Then

u′ = λAeλx + Beλx + λBxeλx u″ = λ2Aeλx + 2λBeλx + λ2Bxeλx

Hence

au″ + bu′ + cu = A[aλ2 + bλ 1 + c]eλx (4.3) +B[2aλ + b]eλx (4.4) +Bx[aλ2 + bλ + c]eλx (4.5)

But

aλ2 + bλ + c = 0 ⇒ (4.3) and (4.5) vanish

and

λ = − b 2a ⇒ (4.4) vanishes

□

Forced linear second order ordinary differential equations
au″ + bu′ + cu = f(x)

In practice solved by three methods:

1.
Undetermined coefficients: the method we will use. Easy but only works for simple cases.
2.
Transforms:

Fourierû(ω) = ∫ −∞∞e−iωxu(x)dx
Laplaceû(p) = ∫ 0∞e−pxu(x)dx

Widely used in science and engineering.

3.
Variation of constants / Green’s functions
u(x) = ∫ −∞∞G(x − y)f(y)dy

very general (extends to a = a(x) etc. but hard to use (need to find G ).

Basic idea for systematic substitution

1.
Find most general solution possible of the equation
aq″ + bq′ + cq = 0

q : CF: complementary function.

2.
Find a particular solution of the equation
ap″ + bp′ + cp = f(x)

p : PI: Particular integral.

3.
Set u = p + q .

We will focus on finding the particular solution for some simple cases of f(x) .

Particular integrals involving exponentials

Suppose

au″ + bu′ + cu = γeμx, (4.6)

γ , μ constants.

Set p = αeμx .

Then

ap″ + bp′ + cp = αeμx(aμ2 + bμ + c)

So p satisfies (4.6) if

αeμx(aμ2 + bμ + c) = γeμx ⇒ α = γ aμ2 + bμ + c

Remark.

This method blows up if aμ2 + bμ + c = 0. In this case eμx solves

au″ + bu′ + cu = 0,

i.e. eμx is part of the complementary function. This is called resonance. In this case we have to use

p = αxeμx

or if μ is even a double root of aλ2 + bλ + c = 0 use

p = αx2eμx.

Remark.

In case of sums of (non-resonance) exponentials, e.g. 

au″ + bu′ + cu = γ 1eμ1x + γ 2eμ2x

use

p = α1eμ1x + α 2eμ2x,

au″ + bu′ + cu = α cos(βx) + γ sin(βx). (4.7)

for some frequency β .

Even if α = 0 or γ = 0 set

p(x) = ℓ cos(βx) + m sin(βx).

Then

ap″ + bp′ + cp = −aβ2ℓ cos(βx) − bβℓ sin(βx) + cℓ cos(βx) −aβ2m sin(βx) + bβm cos(βx) + cm sin(βx) = (−aβ2ℓ + bβm + cℓ) cos(βx) +(−aβ2m − bβℓ + cm) sin(βx) = α cos(βx) + γ sin(βx)

⇒ℓ = (c − aβ2)α − bβγ (c − aβ2)2 + b2β2 m = bβα + (c − aβ2)γ (c − aβ2)2 + b2β2
